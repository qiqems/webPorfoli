La cara responsive

Brifing

Dissenyar al navegador web una cara que reaccioni al so captat pel micròfon del dispositiu.

Procés

Utilitzar la llibreria P5.js per dibuixar, amb codi, els diferents elements que podem trobar a una cara i relacionar la mida, posició o color d'aquests elements al valor obtingut pel volum de so captat pel micròfon del dispositiu.
